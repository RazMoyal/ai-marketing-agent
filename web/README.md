# Web (Next.js)

- URL API מוגדר ב-`NEXT_PUBLIC_SERVER_URL`.
- טוקן JWT נשמר ב-LocalStorage ומצורף אוטומטית.
- RTL, Tailwind.

עמודים:
- `/login`, `/register`
- `/dashboard` – סטטיסטיקות ופידים
- `/settings` – פרופיל + חיבורים
- `/campaigns` – ניהול קמפיינים + "ממן פוסט"
- `/content` – מחולל תוכן
- `/leads` – לידים
- `/competitors` – ניתוח מתחרים
